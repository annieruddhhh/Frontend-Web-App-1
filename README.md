# Frontend-Web-App-1

## About

This is the frontend of our HabitFlow project. It is used to display habits and allow users to add new habits.

---

## Features

* Home page to view habits
* Add Habit page
* Simple and clean UI

---

## Files

* index.html → main page
* add-habit.html → page to add habits
* home.js → handles home page logic
* add-habit.js → handles adding habits
* style.css → styling

---

## How to Run

Open index.html using Live Server.

---

## Note

Make sure backend is running, otherwise this message will show:
"Could not load habits. Is Flask running?"
